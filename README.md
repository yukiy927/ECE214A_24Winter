## Usage

You can see the overview of our pipeline, below is the result we already run with the given datasets.

On the training datasets, we get 100% accuracy.

- Test Clean Acc = 0.814317673378076

- Test Noisy Acc = 0.7521613832853026

Then go to the second module - Test Demo, we can test the blind datasets right under this module. And here are the simple steps for your reference.

1. Run the **GTCC extraction** module get `y_train`,`y_test_clean`,`y_test_noisy` and also all the GTCC features. Then save them in csv files.

2. Run the **MFCC+Chroma+tempo extraction** module get all the MFCC+Chroma+tempo features. Then similarly, save them in csv files.

3. Then **concatenate** two selected features and saved under `train_cg_feat_df`,`test_clean_cg_feat_df`,`test_noisy_cg_feat_df`.

4. Run the **fine-tuned XGBoost model** and get the blind data test accuracy.
